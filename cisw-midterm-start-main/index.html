Web Programming Certificate

Class 1

HTML, CSS, & JavaScript

Plan, program, implement, publish, and maintain websites using HTML, CSS, & JavaScript. Class 2

Principles of Object Oriented Design

This course offers a language-independent introduction to programming logic using object-oriented principles. It introduces programming concepts, such as structure, decision making, looping, arrays, and methods, and it enforces good style and logical thinking. Advanced chapters cover object-oriented issues such as event-driven programming, exceptions, and UML.

Class 3

Windows Operating System

Windows skills for microcomputer support specialists using built-in applets; working with Explorer to organize your disk; optimizing Windows performance; installing Windows hardware and software; using screen capturing and printing; editing Windows system files; connecting to the Internet.

Class 4

Introduction to Psychology

This course will examine psychological approaches to the study of behavior and mental processes. Topics include the history of psychology, psychological research methods, biological psychology, sensation and perception, consciousness, learning, memory, cognition, intelligence, and language, lifespan development, motivation and emotion, applied psychology (e.g., gender and sexuality and stress and health), social psychology including social and diversity issues, personality, psychological disorders, and psychological treatment.

Class 5

Elementary Spanish

This course is designed to enable the student to understand, read, ask, and respond to simple questions in the simple present and future periphrastic tenses, the present progressive, and to use the regular forms of the preterit tense, in addition to using the vocabulary and grammar listed in the course syllabus. The student will apply these same skills to be able to read and write simple paragraphs in Spanish by the end of the course.
